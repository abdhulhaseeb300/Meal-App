# Meal_App

A Flutter-based Meal Application that allows users to explore, filter, and favorite different meal recipes.  
This project demonstrates modern Flutter practices, including state management, navigation, and clean UI design.

## Features

- Browse a list of meals with images and details  
- Filter meals based on dietary preferences (gluten-free, vegan, vegetarian, etc.)  
- Mark and unmark favorite meals  
- View detailed meal information including ingredients and preparation steps  
- Simple, responsive, and modern user interface  

## Technologies Used

- Flutter  
- Dart  
- Provider or setState for state management  
- Material Design Components  
- Navigation with named routes  

### Prerequisites

Ensure the following are installed on your system:
- Flutter SDK  
- Android Studio or Visual Studio Code  
- A connected device or emulator
